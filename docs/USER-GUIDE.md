# MCU 阅读器 · 无线传书使用指南

> 通过手机蓝牙 + 网页扫码，将电脑/手机上的电子书无线传输到 MCU 阅读器。无需数据线，无需安装 App。

---

## 前置准备

开始之前，请确保：

- **MCU 阅读器** 已开机，电量和蓝牙正常
- **智能手机**（Android 或 iOS，浏览器建议使用 Chrome）
- 确保手机**能正常上网**（首次使用需要加载网页，之后 PWA 缓存可离线使用）

---

## 第一步：手机蓝牙配对 MCU

1. 打开手机 **「设置 → 蓝牙」**
2. 扫描附近设备，找到 MCU 设备（名称类似 `E-Reader-XXXX`）
3. 点击 **配对**

![蓝牙配对界面占位](screenshots/step1-bluetooth-pair.png)

> ⚠️ **注意**：部分 Android 机型首次配对后需在蓝牙设备设置中勾选 **"互联网访问"** 或 **"蓝牙共享网络"**，否则下一步无法正常联网。

---

## 第二步：开启「蓝牙共享网络」

此步骤让 MCU 通过手机蓝牙访问网络。

- **Android**：设置 → 连接与共享 → **蓝牙网络共享** → 开启
- **iOS**：设置 → 蓝牙 → 找到已配对设备 → 开启 **网络共享**

![蓝牙共享网络占位](screenshots/step2-bluetooth-tethering.png)

> ⚠️ **注意**：开启后请保持 MCU 与手机在 **3 米以内**，蓝牙信号过远会导致传书中断。

---

## 第三步：扫码进入传书主页

1. 打开 MCU 上的 **「传书」** 功能，屏幕会显示一个二维码
2. 用手机 **扫描二维码**，浏览器自动跳转到传书主页
3. 页面顶部的 **SN 号会自动填充**，状态显示为 **「已连接」** ✅

![扫码界面占位](screenshots/step3-qr-scan.png)
![传书主页占位](screenshots/step3-home-page.png)

> ⚠️ **如果 SN 未自动填充**：可在顶部输入框中手动输入。SN 格式：字母或数字开头，1-64 位，仅允许字母、数字和连字符(`-`)。MCU 设置中可查看本机 SN。

> **提示**：首次加载页面后，浏览器会缓存页面资源，之后即使手机网络不通，也能通过 PWA 离线访问。

---

## 第四步：上传书籍文件

1. 在 **「上传书籍」** 标签页中，点击上传区域（"拖拽文件到此处，或点击选择"）
2. 在文件选择器中选中要传输的文件 **（支持多选）**
3. 等待进度条走完，文件名旁边出现 ✓ 即上传成功

![上传界面占位](screenshots/step4-upload.png)
![上传完成占位](screenshots/step4-upload-done.png)

### 支持的格式与限制

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| 纯文本 | `.txt` | 文本文件 |
| 电子书 | `.epub` | EPUB 标准格式 |
| 文档 | `.pdf` | PDF 文档 |
| 固件 | `.bin` | 二进制固件 |
| 固件 | `.fw` | 固件镜像 |

> 单文件最大 **500 MB**，超过 512 KB 的文件会自动分片上传，断点续传保障大文件稳定传输。

> ⚠️ **上传区域灰色不可点击？** 说明 SN 尚未验证通过，请确认已完成前三步，顶部状态显示"已连接"。

---

## 第五步：推送书籍到 MCU

1. 切换到 **「书籍列表」** 标签页
2. **勾选** 想要推送的书籍（可多选）
3. 点击 **「推送选中 (N)」** 按钮
4. 等待屏幕顶部弹出提示：**"已推送 N 本书，请在阅读器上进行同步"**

![书籍列表占位](screenshots/step5-book-list.png)
![推送成功占位](screenshots/step5-push-success.png)

> ⚠️ **排序提示**：拖拽书籍左侧的手柄图标可调整排序，拖拽后自动保存，无需手动确认。

> ⚠️ **删除书籍**：点击单本书右侧的删除按钮或批量删除，会弹出确认弹窗——**删除操作不可恢复**。

---

## 第六步：在 MCU 上同步

1. 回到 MCU 设备，打开 **「传书」** 功能
2. 点击 **「同步」** 按钮
3. MCU 自动下载已推送的书籍到本地存储
4. 同步完成后可在 MCU 的 **书架/文件管理** 中查看和阅读

![MCU同步界面占位](screenshots/step6-mcu-sync.png)

## 常见问题

### 上传区域是灰色的，点不了？

SN 号未输入或格式不正确。请检查：
- 扫码后 SN 是否成功自动填入
- 如果手动输入，确认格式：字母/数字开头，1-64 位
- 页面顶部状态是否为「已连接」

### 扫码后 SN 没有自动填充？

- 确认二维码中包含了 SN 信息（如 `https://ereader.fun?sn=SN001`）
- 如果没有，请在顶部手动输入

### 扫描二维码不成功？

- 检查手机相机权限是否开启（浏览器设置中允许摄像头）
- 尝试使用 Chrome 浏览器（支持原生扫码，识别率更高）
- 如果仍然失败，可**手动输入 SN**（MCU 设置中查看）

### 上传文件时失败？

- 检查文件大小是否超过 **500 MB**
- 确认手机网络通畅（蓝牙共享网络已开启）
- 尝试刷新页面后重新上传
- 文件名建议使用英文/数字，避免特殊字符

### 推送后 MCU 同步不到书籍？

- 确认手机 **蓝牙共享网络** 仍然开启
- 确认 MCU 与手机在蓝牙范围内（3 米内）
- 在 MCU 传书功能中**点击同步按钮**后等待片刻
- 如果仍然不行，尝试**刷新书籍列表**（点击刷新按钮）

### 书传到 MCU 上后找不到？

- 查看 MCU 的 **书架** 或 **文件管理** App
- 确认传输的文件格式是 MCU 支持的（TXT / EPUB / BIN）
- 固件文件（BIN / FW）不会出现在书架上，它们在固件管理区

### SN 格式有什么要求？

- **必须**以字母（A-Z/a-z）或数字（0-9）**开头**
- 总共 **1 到 64 位**字符
- **只允许**：字母、数字、连字符（`-`）
- 示例：`SN001`、`Reader-Pro-2024`、`MCU10086`
